# OBS-python-scripts
### Здесь вы можете позаимствовать мои плагины для своих стримов. Пользуйтесь, на здоровье. Предлагайте улучшения и подкидывайте идеи новых плагинов.

---
## Инструкция по установке плагина (любого)
1. Убедитесь, что ваша версия OBS не ниже 21.0.
2. Убедитесь, что у вас установлен Python (подойдет любая версия от 3.6 до 3.12 включительно).
3. В верхнем меню OBS откройте ___Сервис___ --> ___Скрипты___.
4. В открывшемся окне переключитесь на вкладку ___Настройки Python___.
5. Нажмите кнопку ___Обзор___ и укажите путь до папки, куда был установлен Python. Пример: <p style="text-align: center;">___C:/Users/User/AppData/Local/Programs/Python/Python312___</p>
6. Вернитесь на вкладку ___Скрипты___.
7. Нажмите кнопку ➕ в левом нижнем углу экрана.
8. Выберите Python-скрипт, который хотите загрузить. Пример: <p style="text-align: center;">___twitch_notifications_into_telegram.py___</p>
9. Справа появятся поля для настройки скрипта. Заполните их.
10. Можете закрыть окно.
